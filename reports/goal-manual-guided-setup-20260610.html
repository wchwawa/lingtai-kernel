goal-manual guided setup update

结论：本 docs PR 扩写 goal-manual，补上 TUI /goal 触发的 goal.request guided setup 流程：agent 收到 system event 后应读 manual、解释机制、询问目标字段、说明取消语义， 并只在 human 确认后写入 .notification/goal.json。

新增内容

  • /goal 不直接写 goal.json，只向 .notification/system.json 追加 source="goal.request" event。
  • Agent 收到请求后应询问 objective、criteria、id/status、reminder_delay_seconds、constraints。
  • 写入前必须解释：dismiss goal.reminder 只隐藏提醒，不取消 goal。
  • 取消 goal 需要删除 .notification/goal.json，或把 data.status 标为 inactive/cancelled/canceled。
  • 完成/替换 goal 需要标记 done/complete/completed/superseded，或删除/替换文件。
  • 处理完请求后用 system(action="dismiss", channel="system", ref_id="goal.request:<timestamp>") 精确清理该 system event。
